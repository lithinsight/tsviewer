Thin Section Viewer by LithInsight

Source code can be found at the project GitHub Repository. This viewer uses OpenSeaDragon GeoTIFFTileSource , OpenSeadragon CurtainSync and OpenSeaDragon-Scalebar.

View local tiff files of whole slide images

Whole slide images provided by Lithinsight are aligned, scaled and pyramid enabled to allow efficient viewing below. Select all files (eg PPL, XPL, RL) for the slide you wish to view

Usage Notes:

Choose file(s) for the thin section you wish to view. After loading, image visibility can be toggled using checkboxes. If more than one image is selected, the viewer will use curtain mode to allow interactive cursor switching between images. Images of the current viewport with scale can be downloaded as .PNG with the capture button or pressing "p" on the keyboard
